# L'Éternel — Catálogo Editorial de Luxo

Uma página web conceitual e minimalista desenvolvida para apresentar uma coleção de fragrâncias exclusivas. O projeto combina um design editorial sofisticado, tipografia elegante e interações fluidas para proporcionar uma experiência de usuário imersiva e sensorial.

---

## 🚀 Tecnologias Utilizadas

* **HTML5 Semântico:** Utilização de tags estruturais (`<header>`, `<section>`, `<article>`) para garantir melhor acessibilidade e SEO.
* **CSS3 Avançado:** * Layout responsivo e dinâmico com **Flexbox**.
    * Tipografia fluida utilizando a função `clamp()` para adaptação perfeita em qualquer tamanho de tela.
    * Animações baseadas em curvas *Cubic-Bezier* para uma transição suave e natural.
* **JavaScript (Vanilla):** Implementação da **Intersection Observer API**, uma abordagem moderna e de alta performance para disparar animações de *scroll* apenas quando os elementos entram na área visível da tela.

---

## 🎨 Características do Design

* **Estilo Editorial:** Inspirado em revistas de alta costura, utilizando tipografia contrastante (`Playfair Display` para títulos e `Lato` para o corpo de texto).
* **Layout Assimétrico:** Quebra da grade tradicional com alinhamentos alternados e uso intencional de espaço negativo para destacar o produto.
* **Paleta de Cores Premium:** Tons neutros, suaves e detalhes em dourado acetinado (`#C5A880`) que transmitem a sensação de luxo e sofisticação.
* **Responsividade:** Interface totalmente adaptável para dispositivos móveis, reorganizando os blocos em uma leitura vertical contínua.
